# Changelog

All notable public changes will be recorded here. The project follows
[Semantic Versioning](https://semver.org/) and the structure of
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Added

- Native menu-bar dashboard for live realm population, local history, `$WOC` market context, and
  on-demand community data.
- Configurable realm, record, population, market, price-target, and release notifications with
  cooldowns, quiet hours, mute, disable, and permission recovery.
- Accessible charts, keyboard inspection, Audio Graph descriptors, adaptive typography, and
  Reduce Motion/Transparency and Increase Contrast support.
- History export and permanent local-history deletion.
- Deterministic visual-QA state, page, and text-size matrix.
- Universal Developer ID, hardened-runtime, notarization, and DMG release pipeline.
- Independent companion identity and original world-and-signal app icon.

### Security

- Strict market-pair identity validation, bounded network payloads, resilient feed decoding, and
  signed-bundle verification.
