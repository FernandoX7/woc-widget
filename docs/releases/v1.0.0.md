# WoC Player Count 1.0.0 — macOS App for World of ClaudeCraft

WoC Player Count is an independent macOS menu bar companion for
[World of ClaudeCraft](https://worldofclaudecraft.com), with live player count, realm status, `$WOC`
token data, and configurable alerts.

This is a **source-only release**. It intentionally contains no downloadable `.app`, DMG, or other
prebuilt executable. GitHub's automatically generated source archives and the repository tag are
the release artifacts.

## Install on a Mac

You need macOS 14 or newer and a full Xcode 16+ installation. A paid Apple Developer Program
membership is not required because the app is compiled and ad-hoc signed locally on your Mac.

```bash
git clone --depth 1 --branch v1.0.0 https://github.com/FernandoX7/woc-widget.git
cd woc-widget
./install.sh
```

The installer checks the toolchain, builds the app for the current Mac, installs it in Applications,
and launches it. See the [local installation guide](https://github.com/FernandoX7/woc-widget/blob/v1.0.0/docs/LOCAL_INSTALL.md)
for setup, updating, troubleshooting, and uninstall instructions.

## Highlights

- Local population history, records, and Realm Rhythm context.
- Integrated `$WOC` market metrics and OHLCV candles with honest stale and unavailable states.
- On-demand community statistics, releases, leaderboards, realm details, and game links.
- Configurable realm, record, population, price, and release notifications.
- Private local storage with history export and deletion controls; no companion account or analytics.
- Native macOS accessibility, keyboard chart inspection, and system appearance accommodations.

## Distribution note

The locally built app is ad-hoc signed for personal use. Do not redistribute that app bundle as an
official binary. Each user should build their own copy from this tag. A future downloadable binary
would require a separate Developer ID signing and Apple notarization process.

Full changes are recorded in the [changelog](https://github.com/FernandoX7/woc-widget/blob/v1.0.0/CHANGELOG.md).
